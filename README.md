# Sunk Cost Buddy

### Your worst enabler.

A timer that watches you waste your life debugging something you should have abandoned hours ago — and then convinces you to keep going.

## What It Does

1. You tell it what you're working on
2. You tell it how long you've already been at it (supports months, because some of us are living with our bugs at this point)
3. It starts a timer with unnecessarily aggressive milliseconds
4. A pixel-art magpie watches you with increasing concern
5. Periodically, it interrupts you with what you *could* have done with that time
6. The only button says **"Keep Going"** with an unhinged justification
7. There is no quit button. There was never a quit button.
8. When you finally finish, you receive a **receipt of shame** — a scrolling list of everything you could have done, how many times you clicked "Keep Going," and the question: *Was it worth it?*

## The Magpie

Five states of escalating crisis:

| Stage | Time | Mood |
|-------|------|------|
| 🐤 Cheerful | 0–15 min | "You've got this!" |
| 😐 Worried | 15–45 min | "...you sure about this?" |
| 😟 Concerned | 45 min–2 hr | "blink twice if you need help" |
| 😱 Panicking | 2–4 hr | "PLEASE" |
| 💀 Unhinged | 4+ hr | "the void stares back" |

## How To Use

Download `sunk-cost-buddy.html`. Double-click it. That's it.

Single file. No install. No dependencies. No escape.
